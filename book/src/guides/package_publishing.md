# Publishing Packages

How to create, publish, and share Fajar Lang packages.

## Create a Package

```bash
fj new my-package
cd my-package
```

This creates a `fj.toml` manifest and `src/main.fj` entry point.

## Package Manifest (fj.toml)

```toml
[package]
name = "my-package"
version = "1.0.0"
description = "My awesome Fajar Lang package"
authors = ["Your Name"]
license = "MIT"
entry = "src/main.fj"

[dependencies]
fj-math = "3.0.0"
fj-json = "1.0.0"
```

## Publish to Registry

```bash
fj publish
```

This validates the manifest, creates a tarball, and uploads to the local registry.

## Install a Package

```bash
fj install fj-math
```

Downloads and installs the package from the registry.

## Search Packages

```bash
fj search math
```

Searches by name and description.

## Check for Updates

```bash
fj update
```

## Security Audit

```bash
fj audit
```

Checks installed packages against known vulnerability database.

## Lock File (fj.lock)

After installing dependencies, a `fj.lock` file is generated:

```
# fj.lock v3
# Auto-generated by fj — do not edit manually

fj-math=3.0.0 sha256:abc123 registry
fj-json=1.0.0 sha256:def456 registry
```

Commit `fj.lock` to version control for reproducible builds.
