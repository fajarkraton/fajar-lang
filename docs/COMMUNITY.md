## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full guide. Quick summary:

1. Fork the repo
2. Create a branch: `feat/my-feature` or `fix/my-bugfix`
3. Write tests first (TDD)
4. `cargo test && cargo clippy -- -D warnings && cargo fmt`
5. Submit PR with description

## University Partnerships

Fajar Lang is ideal for teaching:
- **Compiler Design** — self-hosted compiler with 4 stages (lex, parse, analyze, codegen)
- **Embedded Systems** — @kernel/@device annotations, bare-metal targets
- **Programming Languages** — effect system, linear types, comptime evaluation
- **AI/ML** — first-class tensor types, autograd, training loops

Contact: academic@fajarlang.org (planned)
