From Source

Install via Cargo (recommended)

cargo install fajar-lang

Install via shell script

curl -sSf https://fajarlang.org/install.sh | sh

Download binary

Download the latest release from GitHub Releases.

tar -xzf fj-x86_64-unknown-linux-gnu.tar.gz
sudo mv fj /usr/local/bin/

Install via Homebrew

brew install fajarkraton/tap/fajar-lang

Install via Cargo

cargo install fajar-lang

Download binary

Download the latest release from GitHub Releases.

tar -xzf fj-aarch64-apple-darwin.tar.gz
sudo mv fj /usr/local/bin/

Install via Chocolatey

choco install fajar-lang

Install via Cargo

cargo install fajar-lang

Download binary

Download fj-x86_64-pc-windows-msvc.zip from GitHub Releases and add the extracted directory to your PATH.

Build from source

git clone https://github.com/fajarkraton/fajar-lang.git
cd fajar-lang
cargo build --release
# Binary at target/release/fj

Build with LLVM backend

# Install LLVM 18 first:
# Ubuntu: sudo apt-get install llvm-18-dev libpolly-18-dev libzstd-dev
# macOS:  brew install llvm@18

cargo build --release --features llvm

Requires Rust 1.87+ (stable). See CONTRIBUTING.md for full build instructions.
